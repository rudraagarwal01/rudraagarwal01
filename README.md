# 💫 About Me:
I'm a Computer Science student at the University of Maryland, concentrating in Machine Learning with minors in Business and Information Risk Management, Ethics, and Privacy. I'm drawn to the tech industry because building things has always felt like the most direct way to turn curiosity into meaningful impact.  💡 My PhilosophyI gravitate toward complex problems where the "right" answer isn't obvious—where you have to test, learn, and iterate until a solution truly works for real people. Working in technical roles has taught me that progress is usually made in the unglamorous moments: debugging a driver issue, tracing a network configuration, or cleaning up a backend integration.  I'm not chasing perfect code on the first try; I'm chasing clarity, reliability, and systems I can explain and maintain. Through leading peers and building community tools, I've seen firsthand that technology is only truly "smart" if it's usable, inclusive, and built with empathy.🚀 What I'm Building & DoingEnterprise AI Integration: As a Software Engineering Intern at Fannie Mae, I engineered an AI intake portal using React and FastAPI, integrating Claude 4.5 via AWS Bedrock to streamline division-wide workflows.  Context-Aware Applications: At Apollonian.AI, I developed a chat application processing 1,000+ real-time AI messages daily using React, Firebase, and OpenAI/Hugging Face APIs.  Data Pipelines & RAG: During my time at Mphasis, I designed Neo4j-backed vector embedding pipelines to transform unstructured documents for robust semantic search.  Personal Projects: I love bringing ideas to life. I've built AI EchoMail, a GPT-4 and FastAPI-powered adaptive email generator; DocFlow, an automated document classification pipeline; and other tools like AuthGuard Extension, Fitness Genius, and MealMatch.  🛠️ Tech StackLanguages: Python, Java, C/C++, TypeScript, JavaScript, SQL, Swift, Dart, MATLAB, HTML/CSS  Frameworks & Libraries: React, Node.js, FastAPI, Flask, PyTorch, Scikit-learn, OpenCV, SwiftUI  Cloud & DevOps: AWS (Bedrock, SageMaker, Textract, Lambda, S3, EC2), Docker, Terraform, Azure, Linux  Databases & Tools: PostgreSQL, Neo4j, Git, GitHub, Postman, JupyterLab, Xcode
